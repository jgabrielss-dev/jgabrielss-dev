### 💻 Projetos de Engenharia em Destaque

* ⚙️ **[Agrosantos Data Pipeline & ETL](#)**
  Motor Python/Streamlit hospedado em nuvem que ingere planilhas brutas de ERP, higieniza o inventário via regras de Regex armazenadas no banco de dados e executa web-crawlers visuais autônomos para buscar mídias de produtos faltantes.
